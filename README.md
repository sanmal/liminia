# 涯庭 / Liminia

Text-based idle RPG with autonomous NPC simulation.

## Development

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Build for production
bun run build
```

## Documentation

Specifications are managed in Obsidian vault and accessed via MCP.
